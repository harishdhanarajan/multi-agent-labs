# Python MySQL Connection Utility

This project provides a simple example of how to establish a connection to a MySQL database using Python. It includes a utility function for connecting to MySQL, a configuration class for managing database settings, and an application that demonstrates these functionalities.

## Features
- Establishes a connection to a MySQL database using specified configurations.
- Retrieves database configurations from environment variables or uses default values.
- Demonstrates the ability to handle connection success and failure scenarios.
