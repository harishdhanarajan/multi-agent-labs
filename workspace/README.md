# Python MySQL Connection Utility

This project provides a simple example of how to establish a connection to a MySQL database using Python. It includes functions for database operations, arithmetic calculations, and utilities for string manipulations.

## Features
- Establishes a connection to a MySQL database using specified configurations.
- Performs basic arithmetic operations (addition, subtraction, multiplication, division).
- Utility functions for greeting and string manipulation.
- Retrieves database configurations from environment variables or uses default values.
- Demonstrates the ability to handle connection success and failure scenarios.
